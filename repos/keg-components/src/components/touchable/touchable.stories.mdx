### With States
You can have control over the states by passing in `children` as a function.
It gives you access to these 3 states: **pressed**(`web, native`), **hovered**(`web`), **focused**(`web`). The value reflects the current state of the component
```js
<Touchable
  children={({ pressed, hovered }) => {}
/>
```
